Grand Canyon: A Polaroid Story

(Or a Photo Story on Mobile)

For Grand Canyon: A Polaroid Story, my goal was to make a responsive website of images with captions resembling what a Polaroid photo looks like.

I wrote the HTML and CSS from scratch to make the images appear like Polaroid photos. I used media queries to control the appearance of the photos, so if a user is on a mobile device, the photos appear with captions, but no Polaroid-like border. Also, the title of the website changes from A Polaroid Story to A Photo Story.

Another goal was to show how much fun this trip with my family was!
